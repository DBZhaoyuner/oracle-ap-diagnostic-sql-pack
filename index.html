6. SQL Diagnostic Pack / SQL 诊断包

6.1 Unpaid Invoice Analysis / 未付款发票分析

SELECT invoice_id,
       invoice_num,
       vendor_id,
       invoice_amount,
       payment_status_flag,
       invoice_date
FROM ap_invoices_all
WHERE payment_status_flag <> 'Y'
ORDER BY invoice_date DESC;

6.2 Duplicate Invoice Check / 重复发票检查

SELECT invoice_num,
       vendor_id,
       COUNT(*) AS duplicate_count,
       SUM(invoice_amount) AS total_amount
FROM ap_invoices_all
GROUP BY invoice_num, vendor_id
HAVING COUNT(*) > 1
ORDER BY duplicate_count DESC;

6.3 Invoice Hold Analysis / 发票暂停分析

SELECT aia.invoice_num,
       aia.invoice_amount,
       aha.hold_lookup_code,
       aha.hold_date,
       aha.release_lookup_code
FROM ap_invoices_all aia
JOIN ap_holds_all aha
  ON aia.invoice_id = aha.invoice_id
WHERE aha.release_lookup_code IS NULL;

6.4 Supplier Validation / 供应商设置检查

SELECT vendor_id,
       vendor_name,
       enabled_flag
FROM ap_suppliers
WHERE enabled_flag <> 'Y';

5

Document evidence / 记录证据
Save query, result screenshot, business explanation, and interview story.
