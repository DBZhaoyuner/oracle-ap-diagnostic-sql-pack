# Oracle AP Diagnostic Dashboard V2

## 中文项目名
Oracle 应付账款诊断仪表盘 V2

## Purpose
This is an interactive React portfolio project for Oracle EBS Systems Analyst, ERP Business Analyst, and Financial Systems Analyst roles.

## 目的
这是一个可交互的 React 作品集项目，用于展示 Oracle EBS Systems Analyst / ERP Business Analyst / Financial Systems Analyst 相关能力。

## Features
- Invoice search
- Duplicate invoice detection
- Payment hold analysis
- Simulated AI Oracle AP Assistant
- SQL diagnostic pack
- Oracle AP data model
- STAR interview story generator

## 技术能力证明
- Oracle AP functional knowledge
- Oracle EBS table logic
- SQL diagnostic thinking
- Root cause analysis
- Business user communication
- React portfolio development
- GitHub Pages deployment

## Run Locally

```bash
npm install
npm run dev
```
